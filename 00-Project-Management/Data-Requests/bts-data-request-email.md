We would greatly appreciate your assistance in obtaining the following data:

Specifically, we are looking for the equivalent of:
- October2020TransBorderRawData.zip
- November2020TransBorderRawData.zip
- December2020TransBorderRawData.zip

**2. Year 2023 — September, October, November, December**
All three dataset tables (DOT1, DOT2, DOT3) for these months are unavailable. The download page has monthly ZIP files through August 2023 but no files for Sep–Dec 2023.

**3. Year 2009 — DOT2 (State x Commodity) table only**
The Revised 2009 Public Data ZIP contains DOT2 year-to-date data only through August (dot2_ytd_0809.csv). The DOT1 and DOT3 YTD files for December 2009 are present and complete, but DOT2 appears to have Sep–Dec data missing. Could you provide the complete DOT2 year-to-date file through December 2009?

For reference, the standard file formats we are working with are:
- DOT1 (Table 1 — State x Port): dot1_ytd_MMYY.csv
- DOT2 (Table 2 — State x Commodity): dot2_ytd_MMYY.csv
- DOT3 (Table 3 — Port x Commodity): dot3_ytd_MMYY.csv

If the full monthly ZIP files are available, those would be ideal. Otherwise, any format containing the record-level data with monthly granularity would work for our purposes.
